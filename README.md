## Features
1. Simple and easy to use Ui
2. Nullsafe and proper validation and sanitization of the URLs i.e adding https:// or trimming URL
3. Follws modern MVVM clean aritecture
4. Uses RoomDatabase for caching URL
5. Uses retrofit and beeceptor for remote upload and local deletion of URls
6. Easy logic management of close and back
7. Uses Hilt DI insted of manual injection

## Tech Stack
Languages:- Kotlin
Libraries:- Jetpack Compose(UI), Retofit(Network Calls), 
Database:- RoomDatabase (Caching URLs)
Version Control System:-GitHub
